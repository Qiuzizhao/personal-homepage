# 个人主页 - 设计规范 (Current State)

## 1. 概念与愿景

一个充满个人温度与呼吸感的极简手写风（Hand-drawn Minimalist）主页。通过引入手绘线条、纸张纹理、手写字体和不规则的元素，打破传统网页的冰冷感，仿佛是一本随意涂鸦的旅行日记和手账。

## 2. 设计语言

### 美学方向
**极简手写风 (Hand-drawn Minimalist)** — 放弃了绝对的直线和完美的圆角，大量使用 `border-radius` 不规则参数、随机轻微倾斜 (`rotate`) 以及素描质感的阴影。背景带有微妙的 SVG 噪点，模拟真实的纸张纹理。

### 色彩方案
```css
--bg: #FDFCF8;                /* 暖黄色的纸张白 */
--bg-card: #FFFFFF;           /* 卡片纯白 */
--text: #2C2A29;              /* 接近黑色，模拟碳素墨水 */
--text-muted: #75716C;        /* 褪色的墨迹 */
--border: #E8E4D9;            /* 边框色 */
--border-light: #F2EFEB;      /* 浅边框色 */
--primary: #2C2A29;           /* 强调色以黑色线条为主 */
--primary-hover: #000000;
--primary-light: #F5F3EC;
```

### 字体排印
- **中文手写体**：`Zhi Mang Xing`（行书风格，用于大标题、导航和按钮）
- **英文手写体**：`Caveat`（随意、自然的英文手写，用于副标题、Logo、日期）
- **等宽字体**：`JetBrains Mono`（技术代码相关）

### 空间与圆角
- **间距系统**：2rem (基础) / 6rem (大区块) / 10rem (超大区块)
- **手绘圆角系统**：利用多个百分比或像素值（如 `2px 15px 4px 12px`）创造出用笔画出的不规则矩形或圆形。
- **最大内容宽度**：1000px（手账风格不需要太宽的版面）

### 动效哲学
- **进场动画 (Reveal)**：上浮淡入时增加轻微的随机旋转角度（如 `-1deg`），模拟贴纸被贴上去的感觉。
- **悬停交互 (Hover)**：
  - 卡片与图片：悬停时除了上浮，还会伴随旋转角度的复原或增加，仿佛拿起了这层纸。
  - 链接与按钮：背景荧光笔高亮动画、手绘下划线显示。
  - 时间轴节点：悬停时放大并轻微旋转，像图钉被拨动。

## 3. 页面结构与手绘元素细节

### 页面流程
```
[导航栏 Navbar]
├── 手写 Logo (倾斜)
└── 链接 (Hover 时显示 SVG 手绘波浪下划线)

[英雄区 Hero]
├── SVG 绘制的手绘网格纸背景
├── 巨大的手写中文单字 (带墨水晕染阴影)
├── 个人 tagline
└── 手绘边框的 CTA 按钮

[项目区 Projects]
├── 标题 (带 SVG 绘制的波浪下划线)
├── 顶部 SVG 手撕纸张边缘分隔线
└── 项目卡片网格
    ├── 卡片带有随机轻微倾斜
    ├── 内部图片带有复古 Sepia 滤镜
    └── 链接悬停时呈现黄色荧光笔高亮效果

[旅行区 Travel / 世界]
├── 顶部与底部使用手绘虚线 (dashed border) 分隔
└── 水平横向滚动时间轴
    ├── 时间节点 (不规则圆点，模拟图钉)
    ├── SVG 绘制的虚线连接线
    └── 照片采用正方形拍立得风格，顶部带有半透明胶带效果，照片本身带有轻微褪色滤镜
```

## 4. 技术栈与工程架构

### 核心技术
- **纯 HTML5 + CSS3 + Vanilla JavaScript**
- **CSS 变量 (Custom Properties)** 
- **内联 SVG 数据 URI (Data URIs)**：大量使用 `background: url("data:image/svg+xml,...")` 来绘制手绘波浪线、网格背景、纸张撕裂边缘等，无需加载外部图片文件。

### 依赖资源
- **图标**：`Lucide Icons` (由于其线条风格与手绘风非常契合，直接修改 stroke 粗细即可完美融入)
- **字体**：`Google Fonts` (Zhi Mang Xing, Caveat)
- **图片**：项目本地图片 + Unsplash 占位图
