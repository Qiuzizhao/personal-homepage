# 个人主页

一个简单、温暖的个人主页，展示我做过的一些有趣的项目。

## 风格

清新绿色奶油风 — 温暖的奶油色底色搭配柔和的灰绿色调，像春天午后的一杯抹茶。

## 技术栈

- HTML5 + CSS3 + Vanilla JavaScript
- Google Fonts (Playfair Display, Cormorant Garamond, Source Sans 3)
- Lucide Icons

## 本地运行

```bash
# 直接用浏览器打开
open index.html

# 或者用任意静态服务器
npx serve .
```

## 部署

项目可直接部署到 GitHub Pages。推送到 GitHub 后，在仓库 Settings → Pages 中开启即可。

## 项目结构

```
personal-homepage/
├── index.html      # 主页面
├── css/
│   └── style.css  # 样式
├── js/
│   └── main.js    # 交互逻辑
├── images/         # 项目截图
├── SPEC.md        # 设计规范
└── README.md
```
